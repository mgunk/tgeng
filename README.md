#  ТГ-чат-бот «Обучалка английскому языку»» по курсу «Базы данных

Бот для изучения английских слов методом карточек.


## Функционал
- Тренировка в формате карточек (выбор перевода).
- Удаление слов из личного списка (включая предустановленные).
- Добавление собственных пар слов.
- Визуальная индикация ошибок при выборе ответа.

## Схема базы данных
В проекте используется PostgreSQL и SQLAlchemy. Модель включает 3 таблицы:
1. `Users`: хранит ID чата пользователя.
2. `Words`: общий словарь (слово и перевод).
3. `UserWords`: таблица связей, которая отслеживает, какие слова пользователь удалил или добавил лично.

## Установка и запуск
1. Клонируйте репозиторий.
2. Установите зависимости: `pip install -r requirements.txt`.
3. Создайте файл `.env` и укажите там ваш `BOT_TOKEN` и `DSN`
### `BOT_TOKEN=ваш_токен`
### `DSN=postgresql://login:password@localhost/youdb`

4. Чтобы создать ваш `BOT_TOKEN` в телеграм, перейдите в чат `@botfather` 
5. Запустите бота: `python main.py`.
